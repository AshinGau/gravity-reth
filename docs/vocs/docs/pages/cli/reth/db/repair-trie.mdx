# reth p2p bootnode

Bootnode command

```bash
$ reth p2p bootnode --help
```
```txt
Usage: reth p2p bootnode [OPTIONS]

Options:
      --addr <ADDR>
          Listen address for the bootnode (default: ":30301")

          [default: :30301]

      --gen-key <GEN_KEY>
          Generate a new node key and save it to the specified file

          [default: ]

      --node-key <NODE_KEY>
          Private key filename for the node

          [default: ]

      --nat <NAT>
          NAT resolution method (any|none|upnp|publicip|extip:\<IP\>)

          [default: any]

      --v5
          Run a v5 topic discovery bootnode
========
